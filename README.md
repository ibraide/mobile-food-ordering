# Mobile Restaurant Menu

## Getting Started

Install the dependencies and run the project locally:

```bash
npm install
npm start
```

## Deploying to Netlify

This project includes a `netlify.toml` file, so Netlify can detect the production build settings automatically.

- Build command: `npm run build`
- Publish directory: `dist`

Before deploying, run:

```bash
npm run build
```
